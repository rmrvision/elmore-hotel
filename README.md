# El More — отель: сайт + админка

Премиальный одностраничный сайт-визитка с админкой для редактирования всего контента.
Без БД — данные хранятся в JSON-файле (локально или в S3-бакете).

- **Сайт:** Node.js + Express + EJS, кастомная типографика (Cormorant + Manrope)
- **Админка:** 10 разделов, медиатека с загрузкой, автоматические бэкапы
- **Авторизация:** bcrypt + HTTP-only session cookie
- **Хранилище:** S3 (продакшен) или файловая система (разработка)

---

## Локальный запуск

```bash
npm install
cp .env.example .env
npm run setpass              # задать пароль админки
npm start
```

Сайт: http://localhost:3000 · Админка: http://localhost:3000/admin

В локальном режиме S3 не нужен — данные пишутся в `content.json`, `uploads/`, `backups/`.

## Что редактируется в админке

10 разделов в боковой панели:

1. **Hero** — заголовки, мета, фото
2. **Философия** — цитата, параграфы, статистика
3. **Номера** — категории, специфика, фото (Эконом / Стандарт+ / Люкс)
4. **Галерея** — мозаика фото + цитаты
5. **Удобства** — 8 плиток
6. **Бронирование** — поля формы, категории
7. **Контакты** — адрес, телефон, email
8. **Подвал**
9. **Шапка / Навигация**
10. **Медиатека** — все загруженные файлы

Изменения сохраняются по кнопке «Сохранить», есть «Откатить» и защита от закрытия вкладки.

## Безопасность

- Пароль — bcrypt (12 раундов), хранится только хэш
- Сессия — HTTP-only cookie, sameSite=lax, secure в продакшене
- Все админ-роуты под `requireAuth`
- Загрузки — только изображения, лимит 10 МБ
- `.env` в `.gitignore`, в репозитории не лежит
